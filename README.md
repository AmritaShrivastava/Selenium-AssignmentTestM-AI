# Selenium-AssignmentTestM-AI

## Overview
This project automates Amazon product search using Selenium WebDriver and TestNG.
Tests are executed in parallel on LambdaTest Cloud.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- LambdaTest Cloud

## Test Scenarios
### Test Case 1
- Navigate to Amazon.in
- Search for iPhone
- Add the first listed iPhone to cart
- Fetch and print the price

### Test Case 2
- Navigate to Amazon.in
- Search for Samsung Galaxy device
- Add the device to cart
- Fetch and print the price

## Parallel Execution
- Test cases are executed in parallel using TestNG
- Executed on LambdaTest Cloud (Windows 11, Chrome)
